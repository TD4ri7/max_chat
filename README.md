# MAX Chat

MAX Chat — это веб-приложение для общения, которое позволяет пользователям взаимодействовать друг с другом в режиме реального времени[cite: 3]. Приложение поддерживает глобальный чат, приватные диалоги и групповые чаты[cite: 3]. Проект разработан с учетом принципов распределенных приложений[cite: 3].

**Составитель:** Димитрий Марковский[cite: 3]  
**Учебная группа:** JPTV23[cite: 3]

## Функции
- **Управление пользователями:** Регистрация, безопасный вход (на основе сессий) и выход из системы[cite: 3].
- **Управление профилем:** Изменение личных данных и загрузка аватара[cite: 3].
- **Чаты:** Общая комната (глобальный чат), приватные беседы (1-на-1) и групповые чаты[cite: 3].
- **Сообщения:** Отправка текстовых сообщений и изображений в реальном времени[cite: 3].
- **Модерация:** Пользователи с правами администратора могут удалять неприемлемые сообщения[cite: 3].

## Технологии
- **Сервер (Backend):** Node.js, Express.js[cite: 3]
- **База данных:** MongoDB (Mongoose)[cite: 3]
- **Клиент (Frontend):** JavaScript (ES6+), HTML5, CSS3, Fetch API, FormData[cite: 3]
- **Аутентификация:** На основе сессий (Session-based Authentication)[cite: 3]

## Установка и запуск

### Предварительные требования
Убедитесь, что на вашем компьютере установлены:
- [Node.js](https://nodejs.org/) (версии 18.x или новее)[cite: 3]
- [MongoDB](https://www.mongodb.com/) (локальная служба или кластер MongoDB Atlas)[cite: 3]

### Пошаговое руководство

1. **Клонируйте репозиторий:**
```bash
   git clone <url-вашего-репозитория>
   cd max-chat
   ```[cite: 3]

2. **Установите необходимые зависимости:**
```bash
   npm install
   ```[cite: 3]

3. **Настройте переменные окружения:**
   Создайте в корневой папке проекта файл с именем `.env` и добавьте в него следующие конфигурации:
```env
   PORT=3000
   MONGODB_URI=mongodb://localhost:27017/maxchat
   SESSION_SECRET=minusalajanevõti123
   ```[cite: 3]

4. **Запустите приложение:**
   В режиме разработки (development):
```bash
   npm run dev
